🐳 Prerequisite

Make sure you have already created a Docker Hub account.

PWD

This post is about “Play with Docker” chrome-extension. Play with Docker(PWD) by itself is a simple, interactive and fun playground to learn Docker. This will give you a throwaway *nix shell, in which you can safely play with docker.

The extension adds a “Try in PWD” button on any Docker Hub image page. Here is a short demo of the same.
