3 simple steps to keep your daily log 📝

May 3, 2019 23:00 · 177 words · 1 minute read productivity

For me, it is important to keep track of daily tasks I do at work. And there are loads of tools 🔧 available to help you do that.

However most of the time, on your work machine / office laptop 💻, you do not have the privilege to install the tool of your choice.

So here, I just wanted to share a little-known simple & neat trick 🔨, that I have been using more than a decade successfully which is built right into the Window’s default Notepad app.

  • Create a new text file ex: dailyLog.txt 📄
  • Add the text .LOG (in uppercase) in the first line of the file, followed by Enter.
  • 💾 Save the file and close it.

Now each time you open the file, Notepad appends the current timestamp ⏰ to the bottom of the file and place cursor below it.

This cool tip was originally posted in one of MSDN blog on July 6 2006, by then a Microsoft employee Derek Dysart, in which he mentioned, this feature has been around since Windows 98.
