This post is about a simple interactive visualization I made of glimpses analytics data, which reflects the global distribution of glimpses users with maximum sessions.

Thanks to Tableau Public and github-pages, I found these two solutions immensely helpful in creating, embedding and publishing viz.
Tableau Public is a free software that can connect to a file (Excel, CSV, JSON..) or to a server and create stunning interactive data visualizations for the web.

Please be aware that as the name suggests Tableau public is 100% public and not recommended to sharing private or proprietary data.
